# Student Grading System

A Python script that evaluates students scores and assigns grades.

## How to run

python main.py

## Grading scale
------------------
| Points | Grade |
|--------|-------|
| 0-49   |   2   |
| 50-64  |   3   |
| 65-79  |   4   |
| 80-100 |   5   |
------------------

To pass, student must receive a grade higher than 2.
